## Bar Chart



### Overview
* Bar chart uses bars to show comparisons between categories of data. 
* Bar chart always have two axis. One axis will generally have numerical values, and the other will describe the types of categories being compared.
* These bars can be displayed horizontally or vertically.
### Use case include:
* True Product usage
* Monthly Data Usage
* Mobile Banking Usage

### Anatomy


<Story id="chart-line--basic" />

* ** Visualisation Title **
  * Titles should give users a clear overview of what a data visualisation is about
  * Title should convey information and make a point
  * Title should short as many as possible, could be two lines if necessary
* ** Subtitle/Key metric ** 
* ** Bar ** - Bar size represent value of data
* ** Category Label ** - Show data type name or category name
* ** Axis ** - Axis showing data type
### Tooltip and value


<img src={img} />

* Tooltip can be shown Percentage and Absolute
* How to display Absolute Value
  * 0 - 999 
  * 1K - 999K 
  * Absolute Value can be shown one decimal e.g. 1.6K
* Percentage display without decimal
* Value can be both Percentage and Absolute
* Multiple absolute value can be shown in tooltip
